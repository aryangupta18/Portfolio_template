# Portfolio (Multi-page)

Multi-page glassmorphism portfolio built with HTML, CSS, and vanilla JavaScript.

## Pages
- `index.html` — Home
- `projects.html` — Projects + tag filters
- `education.html` — Education timeline + skills
- `contact.html` — Contact form UI (front-end only)

## Run locally
Open `index.html` in your browser.

Tip: for best results (and to avoid some browser restrictions), run a simple local server:

### PowerShell
```powershell
python -m http.server 5173
```

Then open `http://localhost:5173`.

## Background image
Add a background image at:
- `assets/images/bg.jpg`

If you don’t add one, the site still uses gradient backgrounds.
